dans cette Projets je representer gestion de Stock qui caractérise par:
1)Fonctionnalités principales
- Gestion des produits Ajout, Affichage.
- Gestion des dates avec validation.
- Traitement des exceptions pour une saisie sécurisée.
- Persistance des données dans un fichier texte Stock.txt.
2)Le projet est structuré comme suit :
- Date.cpp/h : Gestion des dates.
- StockItem.cpp/h : Gestion des entités de stock.
- Inventory.cpp/h : Gestion de la collection des produits.
- Exception.cpp/h : Gestion personnalisée des erreurs.
pour cette Projets est developper par language de programation Back-end C++ et utilisation de POO.
